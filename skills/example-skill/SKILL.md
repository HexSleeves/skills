# Example Skill

This is a starter skill. It demonstrates the `SKILL.md` format expected by `npx skills`.

## When to Use

- When you want to see how a skill is structured
- As a template for creating your own skills

## Steps

1. Copy this directory under `skills/`
2. Edit the YAML frontmatter (`name`, `description`) in `SKILL.md`
3. Replace the body with your own instructions
4. Push to GitHub
5. Run `npx skills add HexSleeves/skills --skill <your-skill-name>` to install
